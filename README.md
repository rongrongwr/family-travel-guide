# family-travel-guide: 亲子游学伴学手册生成器

**V4.2** - 为家长生成17页A4可打印伴学手册，让旅行更有意义。

## 简介

一个 AI Skill，输入景点名称和孩子年龄，自动生成专业的亲子伴学手册。参考 DK 儿童百科、国家地理儿童版的设计理念，采用多样化版式、丰富阅读节奏、充足留白。

## 核心特性

- ✅ **17页A4可打印** - 直接打印，断网可用
- ✅ **6种页面模板** - 整版大图/上下结构/时间轴/大数字/九宫格/漫画互动
- ✅ **7种阅读节奏** - 震撼开场/知识探索/数据冲击/故事带入/互动游戏/信息图/总结回顾
- ✅ **IP导览员** - 每页陪伴，降低知识传递的严肃感
- ✅ **丰富互动** - 找一找/数一数/猜一想/连一连/涂色/测验/打卡
- ✅ **家长指南** - 预约方式/推荐路线/必带物品/互动建议

## V4.2 新增

- **内容一致性自查四规则** - 确保知识点在不同位置表达一致
- **传统文化知识点标准核实** - 五行/八卦/二十四节气等先查标准答案
- **专有名词体系边界检查** - 避免混入不同体系的内容
- **简化表述边界标注** - 为通俗说法加准确限定

## 使用方式

```
加载 family-travel-guide 技能，景点=故宫，年龄=10岁
```

## 质量保证

生成后自动执行以下检查：

- 图片可靠性（Base64内嵌，禁止外部URL）
- 测验页隔离（答案在P17）
- 打卡地图导航性（含方位示意图）
- 九宫格完整性（禁止emoji，用手绘风插图）
- 互动区尺寸（涂写区≥8mm）
- IP出场完整性（每页至少1处）
- 时效标注（价格/时间标注时效）
- 内容一致性自查（V4.2新增）

## 视觉基因

每个景点提取专属视觉基因：

- **主色1** + **辅助色2** + **背景色**
- **纹样** - 回纹/云纹等传统纹样
- **IP导览员** - 景点文化强关联角色

## 技术实现

- **单文件HTML** - 所有样式内联，可直接打印
- **@page规则** - A4纵向，安全边距15-20mm
- **print-color-adjust** - 保证打印色彩准确
- **Base64图片** - 所有图片内嵌，断网可用
