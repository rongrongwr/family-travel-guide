# family-travel-guide

> 为孩子生成专属的旅行探索手册，让每一次旅行都成为学习之旅

输入景点名称和孩子年龄，即可生成一份精美的亲子旅行探索手册。

## 它是什么

这是一个 AI Skill，帮助家长在旅行前为孩子准备好探索手册。手册像一本正式出版的儿童读物，包含：

- **景点故事** - 用孩子能理解的方式讲解历史和文化
- **互动任务** - 找一找、数一数、涂色、知识测验
- **实用信息** - 家长指南（预约方式、开放时间、推荐路线）
- **打卡地图** - 记录旅行足迹

## 生成的手册

一份12-20页的A4可打印手册，内容包括：

- 封面与目录
- 景点核心亮点
- 知识科普页（"为什么"系列解释）
- 互动探索页（漫画、九宫格、涂色）
- 知识测验
- 打卡地图
- 家长指南

## 使用方法

```
@family-travel-guide {景点} {年龄}
例如：@family-travel-guide 九寨沟 11岁
```

## 设计特点

- 参考DK儿童百科、国家地理儿童版的设计理念
- 水彩手绘风格插图
- 多样化版式，避免视觉疲劳
- 充足留白，适合儿童阅读
- 所有图片内嵌，断网可打印

## 适用年龄

6-12岁儿童

## 打印说明

生成后直接用浏览器打开，Ctrl+P（或Cmd+P）打印即可。
