# family-travel-guide: 亲子旅行探索手册生成器

**V4.3** - 为家长生成17页A4可打印探索手册，让旅行更有意义。

## 简介

一个 AI Skill，输入景点名称和孩子年龄，自动生成专业的亲子旅行探索手册。参考 DK 儿童百科、国家地理儿童版的设计理念，采用多样化版式、丰富阅读节奏、充足留白。

## 核心特性

- ✅ **17页A4可打印** - 直接打印，断网可用
- ✅ **6种页面模板** - 整版大图/上下结构/时间轴/大数字/九宫格/漫画互动
- ✅ **7种阅读节奏** - 震撼开场/知识探索/数据冲击/故事带入/互动游戏/信息图/总结回顾
- ✅ **IP导览员** - 每页陪伴，降低知识传递的严肃感
- ✅ **丰富互动** - 找一找/数一数/猜一想/圈一圈/涂色/测验/打卡
- ✅ **家长指南** - 预约方式/推荐路线/必带物品/互动建议/推荐阅读

## 质量保证

生成后自动执行以下检查：

- 图片可靠性（Base64内嵌）
- 测验页隔离（答案在P17）
- 打卡地图导航性（含方位示意图）
- 九宫格完整性（禁止emoji）
- 互动区尺寸（涂写区≥8mm）
- IP出场完整性（每页至少1处）
- 时效标注（价格/时间标注时效）
- 内容一致性自查
